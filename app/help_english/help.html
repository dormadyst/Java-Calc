UnitED Help Page

Entering Values

Much like any other calculator values must be entered before any operation can be performed. On this product values can be entered using the standard numeric keypad as highlighted in the image above. The numeric keypad works much like it does on any other calculator.

The decimal button, denoted by a '.' is used to enter decimal numbers. Pressing the decimal button places a decimal point in the number currently displayed on the calculator. If there is no number displayed, pressing the decimal button will automatically prefix the decimal point with a zero, forming '0.' This allows you to enter fractional numbers less than one.

Entering Units


Unlike other calculators, this one allows for more flexibility with units. Expressions can be calculated with or without units. Units, if required, can be selected from a dropdown menu, which includes a 'unitless' option for calculations that do not require specific units. Additionally, the dropdown menu provides a blank option where custom units can be input. This functionality is accessible by clicking on the dropdown highlighted in the left image; once clicked, it expands as shown in the right image, allowing a unit or custom input to be selected by clicking. If a custom unit is input from the unit dropdown initially, there are no conversions.

Custom Like Units (With Conversions)


Custom units can also be loaded so that new like units are defined with conversion factors. This is achieved by using the "Add" button underneath the "Units" menu as seen in the left picture above. Then, as seen in the left picture above, the new like unit name is entered, then the type/unit it is to be converted with, and then the conversion factor. For example, to enter the custom unit "furlong" in relation to meters, you would enter "Furlong" as the name, "m" as the type, and 201.168 as the conversion factor because 1 furlong is equal to 201.168 meters.

Selecting Operations


The process of selecting operations functions as would be expected of a traditional calculator. Operations must be selected between the entering of each operand. They can be selected using the buttons in the rightmost column as shown in the image. Each button corresponds to one of the basic operations, represented by the symbol on its face. Additionally, there is now an exponent button which, when pressed, brings up a window allowing users to input both the number and its exponent. This feature enhances the calculator's functionality, making it more versatile for complex calculations.

Evaluating


Now to put it all together. First a left operand with a value and a selected unit must be entered on the lower display. Then an operation must be selected this will send both up to the upper display. Then a second operand with a value and a unit must be entered on the lower display. The to evaluate the expression the "=" button as shown in the image must be selected. After its selection the expression with be evaluated and the expression and the result will be shown in the upper display.

Other Functions


This section summarizes other buttons on the calculator:

  • "+/-" button: Toggles the sign of an operand.
  • "C" button: Clears the current operand.
  • "R" button: Resets the calculator.
  • "." button: Decimal point for entering decimal values.
  • "" button: Backspace to remove characters.
  • ">" button: Expands the history to view previous calculations.

Menu Bar


As shown above on the left picture, under the File option on the menu bar are the following and their functions:

  • Exit: Closes the application.
  • Print Session: Opens a dialog to print the current session's contents.
  • Save Session: Opens a dialog to save the session history to the device.
  • New Window: Opens a new instance of the calculator in a separate window.
And as in the middle picture, under the Preference option on the menu bar are the following and their functions:
  • Edit: Allows customization of the calculator settings, including setting the number of digits displayed, choosing between fixed or maximum digits, and enabling or disabling comma separation.
  • Open: Opens a dialog to load a previously saved session.
  • Save: Saves the current settings and session.
  • Save As: Saves the current preferences to the device for future sessions.
And as in the right picture, under the Help option on the menu bar are the following and their functions:
  • About: Displays information about the application, such as version number and developer details.
  • Help: Opens the help page, providing guidance and answers to common questions about using the calculator.
